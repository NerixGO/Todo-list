# Todo-list
Simple Todo-list for CLI

Use: “td list” to show task list
Use: “td add [name]” to add task
Use: “td done [name]” to do task
Use: “td done [name]” to remove task
Or type “td -h” to see help text

To install paste this into your terminal

```bash
git clone https://github.com/NerixGO/Todo-list.git
cd Todo-list
./install.sh
cd ..
rm -rf Todo-list
```
And then you are free to use To-Do list!

Also you need some CLI utility like:
For Fedora/Ubuntu/Mint
```bash
sudo apt install jq
```
For Arch
```bash
sudo pacman -S jq
```
